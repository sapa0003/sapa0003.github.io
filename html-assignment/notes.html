HTML Tags

Over the past few weeks, we have been learning about different tags used in HTML semantics, below are five commonly used tags

H1 Tag
This tag is used as a header for the page, this is the largest heading tag and is usually reserved for the main title of the page. You may only use this heading once on a page
Emphasis Tag
This is a type of text tag. When used around a word or phrase the text become italacized. We use this to indicate something of importance.
Paragraph Tag
This tag is used for adding a paragraph on text on your page.
Abbreviation Tag
This tag is used for acronyms and abbreviations. In this tag we must also include the true definition of the acronym in order for the reader to identify the term if they are unfamiliar with the acronym.
Quote tag
This tag allows you to add a small quote to your page.
