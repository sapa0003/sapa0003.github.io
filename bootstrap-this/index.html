Get To Know Bootstrap

About

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Mobile First

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Starter Template

The Bootstrap starter template is a great resource to make sure your webpages are equipped with latest tools for design and development. To ensure your page can be optimized for various screen sizes and browsers, we have included HTML doctype, responsive meta tag, box-sizing and reboot.

To start your project, simply copy and paste the template into your code editor. Once set up, place your required elements into the body tag, and with the use of our special classes you will be able to create a CSS framework in no time.

Customizing Elements

Each of Bootstrap’s base class components can be customized using an additional modifier class. When added onto a base class, changes can vary from colour, size, and style.

Click here to learn more.

Colour

Change the colour of backgrounds and components by using Bootstrap colour themes.

Simply add the modifier class to your base class, for example adding .button-danger will give the button element styled below

Size

Change the size of Bootstrap components using size modifier classes such as sm, md, lg (etc.) This is not limited to just Bootstrap components, but also to grid columns to create a responsive layout for various screen sizes.

This button was created using .btn-lg.

Style

Many Bootstrap components have different variations to style components. Create a button with an outline using .btn-outline.
